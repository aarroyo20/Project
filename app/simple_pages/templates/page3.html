Flask

Flask is a Python web framework built with a small core and modularity in mind. It is classified as a microframework because it does not require particular tools or libraries. It has no database abstraction layer, form validation, or any other components, and all those super necessary features are provided by third-party libraries developed by the team behind Flask or other developers from the community.



According to online resources, Flask is used by many companies we are familiar with, such as:

  • Uber
  • Pinterest
  • Netflix
  • AirBnb
  • Lyft
  • Reddit
  • LinkedIn

Therefore, as you can see, even big businesss are utilizing Flask to build and maintain large-scale web applications. This is because Flask is a lightweight framework so even if it might not come with a lot of features, it is a great way to start and scale up.


Flask Blueprint

The key concept behind any Flask Blueprint is that they record operations to be executed later when you register them on an application. A Flask Blueprint is not an application. It actually needs to be registered in an application before you can run it.

PyTest

PyTest is a testing framework that allows users to write test codes using Python programming language. It helps you to write simple and scalable test cases for databases, APIs, or UI. PyTest is mainly used for writing tests for APIs. It helps to write tests from simple unit tests to complex functional tests.
The pytest framework makes it easy to write small, readable tests, and can scale to support complex functional testing for applications and libraries.
