pylint and others


pytest

pytest is the most popular Python testing package. It makes it easy to write, test, and scale to support more complex testing.

To install python, you can run the command: 

pip install -U pytest

Then, to check the version of pytest that is installed, run the command: 

pytest --version
Then, when you run a test, you will see if it fails (and where) or passes:

fail:

pass:


pylint

According to the Pylint user manual, Pylint is a tool that checks for errors in Python code, tries to enforce a coding standard and looks for code smells. It can also look for certain type errors, it can recommend suggestions about how particular blocks can be refactored and can offer you details about the code's complexity.

To run the pylint test, run the command: 

pytest --pylint
You will see if it fails (and where) or passes:

fail:

coverage

After writing tests with your test suite you can generate a coverage report using Coverage.py

To install coverage.py, run the command: 

pip install coverage



You can determine the version by running the command: 

coverage --version
